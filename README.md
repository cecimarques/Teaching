Welcome to some teaching material that can help you in your undergrad life. The idea here is to efficiently transmit concepts that we might see repeateadly in our academic life but quite grasp and/or that we face in an automated, lifeless fashion. I hope you enjoy it ! If you use the material somewhere, please do cite this repository or give the due credit.

Inside each folder you can find .pdfs that I wrote myself which are accompanied by videos explaining concepts. The content in the .pdfs and in the videos are redundant - choose the format you prefer best.
